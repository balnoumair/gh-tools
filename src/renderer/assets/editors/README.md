# Editor action icons (Electron renderer)

These are the **same** editor logos the Raycast extension uses, copied from
`raycast-extension/assets/editors/` (originals from [SVGL](https://svgl.app/)).

The app is dark-themed, so it uses the `*_dark` variants (light-colored fills);
`claude.svg` is full-color. Terminal uses an inline glyph in `EditorStrip.tsx`.

If you change an icon, update it in **both** places to keep the app and the
Raycast extension visually in sync.
