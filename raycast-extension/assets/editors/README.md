# Editor action icons

SVG logos from [SVGL](https://svgl.app/), fetched via [api.svgl.app](https://api.svgl.app).

| File | SVGL source |
|------|-------------|
| `cursor_light.svg` / `cursor_dark.svg` | Cursor |
| `claude.svg` | Claude AI |
| `codex_light.svg` / `codex_dark.svg` | Codex |
| `zed_light.svg` / `zed_dark.svg` | Zed |

Terminal and Finder use Raycast built-in icons. To replace an icon, drop a new SVG here and update `src/lib/editor-icons.ts`.
